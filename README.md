# The Universal Harmonic Matrix: Base-27720 DDS for ESP32

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform: ESP32](https://img.shields.io/badge/Platform-ESP32-brightgreen.svg)](https://www.espressif.com/)

## 📄 Проєкт
Реалізація Direct Digital Synthesis (DDS) фазового акумулятора на основі високоскладеного числа **27720**. Ця архітектура гарантує **абсолютний нульовий фазовий джитер** для ключових нелінійних резонансів (3-ї, 5-ї, 7-ї та 11-ї гармонік), що критично для систем керування плазмою, високоточної нейростимуляції та стабілізації енергетичних мереж.

## 🧠 Математичний фундамент
Стандартні двійкові DDS-системи (наприклад, 16-bit) не можуть генерувати точні раціональні дроби, що призводить до накопичення похибок округлення (фазового шуму) в контурах зворотного зв'язку.
* **Base-27720 = LCM(1..12)** — це найменше число, що ділиться без залишку на всі цілі числа від 1 до 12.
* **7-ма гармоніка:** 27720 / 7 = 3960 (точне ціле число).
* **11-та гармоніка:** 27720 / 11 = 2520 (точне ціле число).

## 📊 Порівняння з 16-bit Binary DDS

| Гармоніка | 16-bit Binary DDS | Base-27720 DDS | Двійкова похибка (LSB) |
| :--- | :--- | :--- | :--- |
| **1/3** | 21845.333... | **9240** | **0.333 LSB** |
| **1/5** | 13107.200 | **5544** | **0.200 LSB** |
| **1/7** | 9362.285... | **3960** | **0.285 LSB** |
| **1/11** | 5957.818... | **2520** | **0.818 LSB** |

## ⚙️ Апаратне забезпечення
* **Мікроконтролер:** ESP32 (будь-яка модель з вбудованим ЦАП).
* **Вивід:** GPIO 25 (вбудований DAC). Також підтримується вивід через ШІМ з інтегруючим RC-фільтром.
* **Оптимізація пам'яті:** 8-бітна таблиця пошуку (LUT) генерується динамічно і займає рівно 27.7 кБ, ідеально поміщаючись у надшвидкий L1-кеш мікроконтролера.

## 🚀 Швидкий старт
1. Встановіть Arduino IDE або PlatformIO.
2. Відкрийте файл `src/main.cpp`.
3. Скомпілюйте та завантажте прошивку на вашу плату ESP32.
4. Підключіть осцилограф до **GPIO 25** та відкрийте Serial Monitor (115200 baud).

## 🔬 Демонстрація роботи
* При старті система проходить математичну валідацію і починає генерувати ідеальну **7-му гармоніку** (3960 кроків на період).
* Кожні 10 секунд відбувається автоматичне перемикання на **11-ту гармоніку** (2520 кроків на період).
* **Phase-Continuous Switching:** Перемикання частот відбувається атомарно, без скидання акумулятора, гарантуючи відсутність розривів або стрибків фази на осцилограмі.
